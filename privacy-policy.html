Application Privacy Policy

SheetCopilot ("the Add-on") is an add-on for Google Sheets that allows users to automate spreadsheet manipulation using the OpenAI API.

Information Collected:

The Add-on does not collect any personal information or API keys from the user. However, it does access the spreadsheet data of the user to send to the OpenAI API for processing. The data sent to the API may include data in the cells of the spreadsheet and the formulas used in the spreadsheet.

Use of Information: The information collected through the Add-on is used solely for the purpose of providing better and more accurate responses to the user. The Add-on does not share or sell any user data with third parties.

Security: The Add-on takes appropriate technical measures to ensure that the user data is protected from unauthorized access, misuse, or loss.

Data Retention: The Add-on needs to temporarily retain spreadsheet states (including column headers, the number of rows, sheet names, and properties of all charts) in the user properties to query ChatGPT for more accurate solutions. The data is deleted after the complete solution is received from the OpenAI API or the user deliberately erases them.

Consent: The user must explicitly consent to the Add-on's data collection and use by accepting this privacy policy.

Contact: If you have any concerns or questions about this privacy policy or the Add-on, please contact us at sheetcopilot@gmail.com. We will do our best to respond to your inquiry in a timely and appropriate manner.

Updates: This privacy policy may be updated from time to time. Users will be notified of any changes to the policy through the Add-on.
