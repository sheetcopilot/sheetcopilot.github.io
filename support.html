Getting started

1. Install from the Google Workspace Marketplace

The SheetCopilot add-on can be found in the Google Marketplace here. Go to the listing and, while logged into the Google account you wish to utilize SheetCopilot from, click the install button.

Next you will be prompted to grant SheetCopilot the permissions required for it to run.

We take great care in only asking for the necessary permissions and do not collect or store any information not essential in running the service. You can view our policy and terms of service if you wish to understand more about how we use your data.

Once all steps have been completed you can find SheetCopilot as a Google Spreadsheet Add-on:

2. Set API Key

SheetCopilot requires an OpenAI API key to run. You can get one by signing up for the OpenAI here. Once you have an API key, you can set it in SheetCopilot by clicking on the "Set API Key" button in the SheetCopilot menu:

3. Start using SheetCopilot

SheetCopilot is now ready to use. You can chat with it by clicking on the "Open chat" button in the SheetCopilot menu:
